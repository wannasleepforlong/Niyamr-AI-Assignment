# NIYAMR AI – 48-Hour Internship Assignment  
**AI Agent for Universal Credit Act 2025 Analysis**  

---

## Overview
This project implements a mini AI agent that reads, summarizes, and analyzes the **Universal Credit Act 2025**, returning a structured JSON report. The AI agent uses Google Gemini AI via LangChain to extract key legal information, summarize the Act, and perform rule-based checks.  

The project completes **Tasks 1–4** as outlined in the assignment brief:  
1. **Extract Text** from the PDF and clean it.  
2. **Summarize the Act** in concise bullet points.  
3. **Extract Key Legislative Sections** into structured JSON.  
4. **Apply Rule Checks** to validate key legal requirements.
